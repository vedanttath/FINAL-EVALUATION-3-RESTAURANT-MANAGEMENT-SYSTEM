# Restaurant Management System

A modern restaurant management system built with React, TypeScript, and Supabase.

## Features

- Real-time order management
- Table reservation system
- Menu management
- Order tracking
- Analytics dashboard
- Take-away order support

## Tech Stack

- Frontend:
  - React
  - TypeScript
  - Tailwind CSS
  - Chart.js
  - Lucide Icons

- Backend:
  - Node.js
  - Express
  - Mongodb
